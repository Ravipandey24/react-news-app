## Run Locally

Clone the project

```bash
  git clone https://github.com/Ravipandey24/react-news-app.git
```

Go to the project directory

```bash
  cd react-news-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
